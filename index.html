  • Interactive Text Editor

    Html, CSS, JavaScript

    Interactive text editor that implements a different method of highlighting(than we were originally trying to do for part of our senior project), where it highlights words by finding specific pixel offsets and drawing the highlight at those locations. This way we could test out some of the functionality that we want without having to mess with editor state management. We made it so that the highlighting gets applied proportionally to the likelihood of a word to be replaced. So the more likely a word to be replaced by the LLM, the darker the highlight would be applied and would only be applied if a words likeliness to be replaced was above a certain threshold. Similarly to the likeliness variable, the backend also generates the most likely alternative replacement word for each highlighted word. We implemented a function that draws that alternative word underneath each highlighted word. Some of the functionality when typing into the editor is still a bit buggy/finicky, but overall it was useful for our project and allows for many more possibilities in the future. (The backend was quickly hacked together by my advisor. I only did added the frontend functionality described)

  • Dutch Bingo

    C#, .NET

    Dutch Bingo uses a directed graph, where each person is represented as a node, and their relationships (such as parent, child, or spouse) are depicted as directed edges connecting these nodes. The program includes several search functionalities: Orphans() identifies individuals with no recorded parents, while Siblings(name) finds siblings of a specified person. Additionally, Bingo(name, name2) calculates the shortest path between two people in the family tree. There are other commands as well, which you can see on the github page or by running the program.
